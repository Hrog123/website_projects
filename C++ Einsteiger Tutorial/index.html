Allgemein

C++ ist eine der bekanntesten Programmiersprachen. Besonders verbreitet ist die Sprache in der Spieleentwicklung und gehört daher für jeden angehenden Spieleentwickler zum Standardportfolio.

C++ ist eine objekt-orientierte Programmiersprache. Dadurch entstehen bei korrekter Anwedung saubere Programme, welche effizient arbeiten können. Das letzte größere Update gabe es im Jahre 2020.

Vorwissen

Dieses Tutorial richtet sich an Einsteiger. Es ist nur wenig Vorwissen von Nöten:

  • Man muss einen Computer im Allgemeinen bedienen können (Ordner öffnen, erstellen, etc.).
  • Man sollte Grundkenntnisse der englischen Sprache besitzen, dieses Tutorial ist zwar auf deutsch gehalten, (fast) alle Begriffe in den Programmen entstammen allerdings der englischen Sprache, es macht den Einstieg erheblich leichter, diese zu verstehen.
Einführung

Um zu beginnen, werden zunächst zwei Dinge benötigt:

  • Ein Texteditor, ich empfehle Visual Studio Code
  • Ein Compiler (Übersetzer), ein Beispiel ist der GCC.

Es macht an dieser Stelle mehr Sinn direkt eine Kombination beider Systeme, eine integriete Entwicklungsumgebung (IDE) zu vewenden. ich empfehle Visual Studio.

Ein erstes Programm

Nun wollen wir auch schon direkt unser erstes, kleines Programm schreiben. Dabei gehen wir den klassischen Weg, und beginnen mit einem Programm, dass uns "Hallo Welt!" ausgibt.

#include <iostream>
using namespace std;

int main() {
    cout << "Hallo, Welt!";
    return 0;
}

Dies speichern wir nun, und compielieren den Code. Als Ausgabe erhalten wir:

Hallo, Welt!

Herzlichen Glückwunsch, wenn das bei dir genauso aussieht, denn dann hast du gerade dein ersten von hoffentlich vielen Programmen in C++ geschrieben! Und wenn nicht, ist das auch kein Problem, schau nach, ob du vielleicht einen Schreibfehler oder eine Klammer vergessen hast. Du kannst auch gerne den hier vorhandenen Code kopieren und dann direkt mit deinem vergleichen.

Erläuterung

Dann wollen, wir uns mal anschauen, warum das passiert ist, was passiert ist:

  • #include <iostream> inkludiert ein "Header File", in diesem sind die Funktionen beschrieben, welche das Programm aufrufen möchte.
  • using namespace std erlaubt es uns die Standard-Funktionen zu nutzen, ohne dies später nochmals extra kennzeichnen zu müssen.
  • Eine freie Zeile, diese ist dem Programm egal, macht es aber für Menschen besser lesbar.
  • int main () { beschreibt eine Funktion, genauer gesagt die Haupt(main)funktion. Der Code in den geschwungenen Klammern wird hierbei aufgerufen {}.
  • cout << "Hallo, Welt!"; sorgt für die Ausgabe von "Hallo, Welt", ohne das vorherige using namespace std; müsste es std::cout << "Hallo, Welt"; lauten.
  • return 0; beendet dann die Hauptfunktion, und damit auch das Programm.
  • } beendet dann auch formal die Hauptfunktion und darf nicht vergessen werden.
Zusammenfassung

Damit hast du nun dein erstes eigenes Programm geschrieben, und (hoffentlich) auch in Ansätzen nachvollziehen können, weshalb es so zu schreiben war, wie du es geschrieben hast. Doch das kann natürlich nur der Anfang gewesen sein, es gibt noch vieles zu entdecken. C++ ist ein sehr großes Feld, dass sich auch mit vielen Jahren Erfahrung nicht immer vollständig überblicken lässt. Lass dich daher nicht von Rückschlägen demotivieren, denn nur Fehler lernt man.

In diesem Sinne hoffe ich, dass du nun einen Einstieg in die Programmiersprache C++ gefunden hast und ich dein Interesse daran wecken konnte.
